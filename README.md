## Description
This repository contains C programming projects completed as part of CSE-13S: Computer Systems and C Programming at UCSC. The projects demonstrate skills in:

- C programming and problem solving  
- Input handling and validation  
- Makefile usage for compilation  
- Writing test scripts  
- Basic data visualization  

Each project is organized into its own subfolder with its own README containing details about the project, files, compilation, and usage.

---

## Projects Included

- **C-Projects-Histogram-Coolness**  
  Contains two projects:  
  1. **Histogram** – Visualizes user input as a histogram with ASCII characters.  
  2. **Coolness** – Calculates a “coolness” score based on temperature and wind speed.  

- **Password Checker**  
  Verifies whether a password meets specified security requirements, including length, uppercase/lowercase letters, numbers, and absence of first/last name.
  
- **LinkedList-Array-Projects**
  Contains two projects:
  1. **growing_arrays** - Implements a dynamic array of records that can add, delete, retrieve, and search records.
  2. **ll_cycle** - Detects whether a singly linked list contains a cycle.
  
- **Data-Structures-Hashtable-Set-LRU**
  Contains several files that implement a Set, Hashtable, and LRU Cache.
