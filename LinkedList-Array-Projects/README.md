## Description
This folder contains two C programming projects completed as part of CSE-13S coursework at UCSC. Both projects demonstrate skills in **C programming, pointer manipulation, dynamic memory management, and algorithm implementation**.  

### Projects Included

1. **Linked List Cycle Detection**  
   - Detects whether a singly linked list contains a cycle using **Floyd’s Tortoise and Hare algorithm**.  
   - Demonstrates pointer traversal, fast/slow pointer technique, and linked list manipulation.  
   - **See [`ll_cycle/README.md`](./ll_cycle/README.md) for details.**

2. **Growing Arrays: Dynamic Record Management**  
   - Implements a dynamic array of records that can **add, delete, retrieve, and search records**.  
   - Demonstrates **dynamic memory allocation (`malloc`/`realloc`), array shifting with `memmove`, and safe index checking**.  
   - **See [`growing_arrays/README.md`](./growing_arrays/README.md) for details.**
